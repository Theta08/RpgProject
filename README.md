# RpgProject
rpg게임 클라이언트 (유로 캐릭터를 써서 소스코드만 올림)

* 개발 환경 
    * 서버 : 윈도우 11
    * boost(v1.83.0)
    * google::protobuf(v3.21.12)
    * 클라이언트 : 윈도우 11, 화면(800 * 600)기준
    * unity (2021.3)
 
### 주요 기능
 * 직업에 따른 플레이어 스킬 변함
 * 일정 체력 이하로 내려가면 보스몬스터 스킬 발동
 * 채팅 기능
 * 스킬 쿨타임
